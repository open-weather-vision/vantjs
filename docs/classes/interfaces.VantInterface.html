Class VantInterface

Interface to any vantage weather station (Vue, Pro, Pro 2). Provides useful methods to access realtime weather data from your weather station's console. The device must be connected serially. To interact with your weather station create an instance of this class using VantInterface.create.

The VantInterface is an EventEmitter. The events fired by the interface are described here.

This interface is limited to station independent features. Use VantPro2Interface, VantProInterface and VantVueInterface for station dependent features.

Static defaultMaxListeners

defaultMaxListeners: number

By default, a maximum of 10 listeners can be registered for any single event. This limit can be changed for individual VantInterface instances using the setMaxListeners method.

To change the default for all EventEmitter instances, this property can be used. If this value is not a positive number, a RangeError is thrown.

Static create

  • create(settings): Promise<VantInterface>

  • Creates an interface to your vantage console (Vue, Pro, Pro 2) using the passed settings (see MinimumVantInterfaceSettings). The device should be connected serially.

    Weather station dependent functionality (e.g. firmware version code for Vantage Pro 2 / Vue) is not supported on this interface. Use VantPro2Interface, VantProInterface and VantVueInterface for station dependent features.

    Parameters

    • settings: MinimumVantInterfaceSettings

      the interface settings

    Returns Promise<VantInterface>

    Example

    const device = await VantInterface.create({ path: "COM3", rainCollectorSize: "0.2mm" });

    const highsAndLows = await device.getHighsAndLows();
    inspect(highsAndLows);

    await device.close();

    Throws

    SerialPortError if the serialport connection unexpectedly closes (or similar)

    Throws

    FailedToWakeUpError if the console doesn't wake up after trying three times

    Throws

    ClosedConnectionError if the connection to the weather station's console is already closed
