Class VantVueInterface

Interface to the Vantage Vue weather station. Is built on top of the VantPro2Interface.

Offers station dependent features like VantVueInterface.getRichRealtimeData, VantVueInterface.getLOOP1, VantVueInterface.getLOOP2, VantVueInterface.isSupportingLOOP2Packages and VantVueInterface.getFirmwareVersion.

defaultMaxListeners: number

Inherited from VantPro2Interface.defaultMaxListeners

  • Defined in src/interfaces/VantInterface.ts:44

By default, a maximum of 10 listeners can be registered for any single event. This limit can be changed for individual VantInterface instances using the setMaxListeners method.

To change the default for all EventEmitter instances, this property can be used. If this value is not a positive number, a RangeError is thrown.

wakeUp

  • wakeUp(): Promise<void>

  • Inherited from VantPro2Interface.wakeUp

    • Defined in src/interfaces/VantInterface.ts:523

    Wakes up the vantage console. This is necessary in order to send and receive data. The console automatically falls asleep after two minutes of inactivity and needs to be woken up again.

    Fires the VantInterfaceEvents.awakening event if the console wakes up.

    throws

    ClosedConnectionError if the connection to the weather station's console is already closed

    throws

    FailedToWakeUpError if the console doesn't wake up after trying three times

    throws

    SerialPortError if the serialport connection unexpectedly closes (or similar)

    Returns Promise<void>

Static create

  • create(settings: MinimumVantInterfaceSettings): Promise<VantVueInterface>

  • Overrides VantPro2Interface.create

    • Defined in src/interfaces/VantVueInterface.ts:30

    Creates an interface to your vantage vue weather station using the passed settings. The device should be connected serially.

    example
    const device = await VantVueInterface.create({ path: "COM3", rainCollectorSize: "0.2mm" });


    const richRealtimeData = await device.getRichRealtimeData();
    inspect(richRealtimeData);

    await device.close();
    throws

    SerialPortError if the serialport connection unexpectedly closes (or similar)

    throws

    FailedToWakeUpError if the console doesn't wake up after trying three times

    throws

    ClosedConnectionError if the connection to the weather station's console is already closed

    Parameters

    • settings: MinimumVantInterfaceSettings

      the settings

    Returns Promise<VantVueInterface>
