DoNothing

DoNothing = 1
  • Defined in src/realtime-data-containers/settings/OnContainerCreate.ts:8

Does nothing. If you choose this option, you must manually start the realtime data container with SmallRealtimeDataContainer.start / BigRealtimeDataContainer.start.

Start

Start = 2
  • Defined in src/realtime-data-containers/settings/OnContainerCreate.ts:14

Starts the realtime data container. See SmallRealtimeDataContainer.start / BigRealtimeDataContainer.start. Doesn't wait for the serial port connection to be opened.

StartAndWaitUntilOpen

StartAndWaitUntilOpen = 3
  • Defined in src/realtime-data-containers/settings/OnContainerCreate.ts:19

Starts the realtime data container and waits until the serial port connection is open. See SmallRealtimeDataContainer.startAndWaitUntilOpen / BigRealtimeDataContainer.startAndWaitUntilOpen.

WaitForFirstUpdate

WaitForFirstUpdate = 4
  • Defined in src/realtime-data-containers/settings/OnContainerCreate.ts:25

Opens the serial connection to the vantage console and waits for the first update (the update can also be invalid). The weather station automatically gets woken up on every update.

WaitForFirstValidUpdate

WaitForFirstValidUpdate = 5
  • Defined in src/realtime-data-containers/settings/OnContainerCreate.ts:31

Default setting. Opens the serial connection to the vantage console and waits for the first valid update. The weather station automatically gets woken up on every update.
