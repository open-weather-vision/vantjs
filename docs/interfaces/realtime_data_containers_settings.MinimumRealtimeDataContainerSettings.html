Readonly path

path: string

Required. The used (serial) path to communicate with your weather station. On windows devices paths usually start with COM followed by the port number, on linux/osx common paths start with /dev/tty followed by the port number.

Optional Readonly units

units?: Partial<UnitSettings>

Optional Readonly updateInterval

updateInterval?: number
  • Defined in src/realtime-data-containers/settings/MinimumRealtimeDataContainerSettings.ts:46

Optional. The interval (in seconds) the container updates itself. Default is 60 seconds.

On update the "update" and the "valid-update" (if the update succeeds) event fires.

A smaller intervall leads to more up-to-date weather data but also more traffic.
