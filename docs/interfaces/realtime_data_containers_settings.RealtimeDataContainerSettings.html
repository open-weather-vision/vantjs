Interface RealtimeDataContainerSettings<SupportedDeviceModels>

Settings for the SmallRealtimeDataContainer and the BigRealtimeDataContainer. MinimumRealtimeDataContainerSettings describes the mimimum required settings to be configured when creating a realtime data container.

Readonly updateInterval

updateInterval: number
  • Defined in src/realtime-data-containers/settings/RealtimeDataContainerSettings.ts:48

The interval (in seconds) the container updates itself. Default is 60 seconds.

On update the "update" and the "valid-update" (if the update succeeds) event fires.

A smaller intervall leads to more up-to-date weather data but also more traffic.
